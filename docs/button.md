# Элемент button  
Используется как самостоятельный элемент, и в составе сложных элементов.

**universal props**
- children
   - text
   - icon
- id
  
**style props**
- variant
- p
- pt
- pr
- pb
- pl
- px
- py
- m
- mt
- mr
- mb
- ml
- mx
- my
- bg
- display
- height
   - minHeight
   - maxHeight
- width
   - minWidth
   - maxWidth
- size
- alignSelf
- justifySelf
- border - Рамка у кнопки. Свитчер.
   - borderSize
   - borderTop
   - borderRight
   - borderBottom
   - borderLeft
   - borderSize - Размер рамки. Принимает цифру.
   - borderRadius - Радиус скругления углов. Принимает цифру.
   - borderColor - Цвет рамки. Принимает строку.
- opacity
- boxShadow
- position
   - top
   - right
   - bottom
   - left
- zIndex

**personal props**
- href
   - targetBlank
