# Элемент carousel
Используется как самостоятельный элемент

**universal props**
- children
    - card
    - image
    - video
- id

**style props**
- variant
- p
- pt
- pr
- pb
- pl
- px
- py
- m
- mt
- mr
- mb
- ml
- mx
- my
- display
- width
   - minWidth
   - maxWidth
- height
   - minHeight
   - maxHeight
- alignSelf
- justifySelf


**personal props**
- autoplay - Автоматическая смена слайдов в карусели. Свитчер
    - autoplaySpeed - Скорость автоматической смены слайдов в карусели. Принимает цифру
